NAME
        et - Enemy Territory

SYNOPSIS
        et [-omg] {pwn}

DESCRIPTION
        We play ET because the game from The Last Starfighter doesn't exist.
        It must always be run with the pwn argument.

SEE ALSO
        wilug(7), flipmode(7)

        If you don't already totally kick ass at ET like the rest of us, read
        Teh Guide:

        http://www.planetwolfenstein.com/enemyterritory/guide.shtml

HISTORY
        We've been playing ET on newlug.org for a while. We don't like 
        strangers, though, so it's password-protected. If we remember the
        password, we'll tell it to you.
